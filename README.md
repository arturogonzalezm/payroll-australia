Important main Tech Specs and Versions:

* Ruby version 2.1.2

* Rails 4.1.4

* SQLite 3

* Cucumber set up

* Selenium set up

* RSpec

Instructions:
 
bundle install
OR
bundle update

THEN

* rake db:migrate
* rake db:seed
* rails s
